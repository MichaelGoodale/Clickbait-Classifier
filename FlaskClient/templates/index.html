What is this?

Clickbait Classifier is a convolutional neural network, which is trained to distinguish clickbait headlines from more sober ones. It is an implementation in tensorflow of Yoon Kim's 2014 paper, Convolutional Neural Networks for Sentence Classification.

It is also open source! You can read the source, and learn more about it on GitHub
